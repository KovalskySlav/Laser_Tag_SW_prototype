# Прототип системы Лазертаг
Проект представляет собой базовый макет двусторонней лазерной коммуникации, реагирующей на движение. Включает в себя код передатчика (лазерного излучателя) и приемника (сенсора попаданий).
## Структура проекта
Laser_tag_Tx - Код передатчика (Передатчик)
Laser_tag_Rx - Код приемника (Приёмник)
### Программные зависимости (Dependencies)
Для успешной компиляции передатчика (TX) в Arduino IDE необходимо установить следующие библиотеки в среду разработки Android IDE:
1. Adafruit LSM6DS3
2. Adafruit Unified Sensor

Код приемника (RX) работает на стандартных функциях Arduino и дополнительных библиотек не требует.
