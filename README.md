По для прототипа системы Лазертаг
Проект представляет собой базовый макет двусторонней лазерной коммуникации, реагирующей на движение. 

Включает в себя код передатчика , используется лазерный излучатель и акселлерометр подключённый по интерфейсу I2C.
В коде реализована калибровка нуля акселлерометра , отправка синхрослова, задержка между "выстрелами", serial monitor для отладки

В приемнике (сенсора попаданий) используется фотодиод и световой индикатор попаданий.
В коде реализована калибровка уровня естественного освещения ,чтение и проверка полученного сообщения(синхрослова), включение светодиода подтверждения,
serial monitor для отладки
